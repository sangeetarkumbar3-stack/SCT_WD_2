# Stopwatch Web Application

This is a simple and interactive Stopwatch Web Application developed using HTML, CSS, and JavaScript.

## Features
- Start, Pause, Reset Stopwatch
- Lap Time Tracking
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
